# 📚 Learn English — Aplikasi Belajar Bahasa Inggris

> Aplikasi Android interaktif untuk belajar Bahasa Inggris dengan cara yang menyenangkan, dilengkapi materi, kuis, dan audio — cocok untuk semua kalangan terutama siswa SD.

---

## 📌 Deskripsi

**Learn English** adalah aplikasi mobile berbasis Android yang dirancang untuk membantu pengguna belajar Bahasa Inggris secara mandiri dan menyenangkan. Dengan tampilan yang menarik dan materi yang terstruktur, aplikasi ini cocok digunakan oleh siapa saja — terutama siswa Sekolah Dasar (SD) yang baru mengenal Bahasa Inggris.

---

## ✨ Fitur Utama

- 📖 **Materi Pembelajaran** — Konten belajar yang terstruktur dan mudah dipahami
- 🔤 **Vocabulary** — Kumpulan kosakata Bahasa Inggris lengkap dengan artinya
- ✏️ **Grammar** — Penjelasan tata bahasa yang sederhana dan mudah dipelajari
- 🎧 **Audio / Pronunciation** — Dilengkapi suara untuk membantu pelafalan yang benar
- 🧠 **Kuis Interaktif** — Latihan soal untuk menguji pemahaman materi
- 🎨 **UI Menarik** — Desain visual yang ramah anak dan nyaman digunakan

## ⚙️ Cara Menjalankan

### Prasyarat
- Android Studio (versi terbaru)
- JDK 8 atau lebih baru
- Android SDK
- Perangkat Android atau Emulator (min. Android 5.0 / API 21)

### Langkah-langkah

1. **Clone repository ini**
   ```bash
   git clone https://github.com/sahal-shinee/App-Learn-English.git
   ```

2. **Buka di Android Studio**
   - Buka Android Studio
   - Klik **"Open"** → pilih folder `App-Learn-English`
   - Tunggu proses Gradle sync selesai

3. **Jalankan aplikasi**
   - Hubungkan perangkat Android via USB, atau buka Emulator
   - Klik tombol **▶ Run** (Shift + F10)
